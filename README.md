# Solidity Docstrings Remix-plugin

This plugin is developed for [Ethereum Remix online IDE](http://remix.ethereum.org/#optimize=false&evmVersion=null).<br>
There is no way to publish it yet. You can only use it locally.<br>
Final effect:
<br>
<img src="images/final_effect.png" width="90%" >

# How to Use

1. install locally

```
npm install
```

2. run it locally on port:8000

```
npm start
```

3. connect your Remix IDE to the Plugin<br>
   step 1 click Button<br>
   <img src="images/step1.png" width="60%" ><br><br>
   step 2 connect to local plugin<br>
   <img src="images/step2.png" width="60%" ><br><br>
   step 3 give it a name and port<br>
   <img src="images/step3.png" width="60%" ><br><br>
   step 4 select a solidity file<br>
   <img src="images/step4.png" width="60%" ><br><br>
   step 5 click the smile face on side panel<br>
   <img src="images/step5.png" width="60%" ><br><br>
   step 6 click any button in the plugin and play with it<br>
